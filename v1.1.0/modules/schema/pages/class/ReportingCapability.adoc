= `ReportingCapability`
:toclevels: 4


+++<font color="#0f0f0f">Definition of one set of reporting capabilities for this monitoring station. The associated EnvironmentalValueSets describe the maximum range of possible environmental values the station is capable of returning.  This attribute is intended primarily to assist a utility in managing its stations. </font>+++
