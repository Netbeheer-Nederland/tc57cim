= `Pnode`
:toclevels: 4


+++A pricing node is directly associated with a connectivity node.  It is a pricing location for which market participants submit their bids, offers, buy/sell CRRs, and settle.+++


[cols="h,3",width=65%]
|===
| Type
| Class

| URI
| `cim:Pnode`


| CIM standard (perspective)
| IEC62325 (Market)



|===
